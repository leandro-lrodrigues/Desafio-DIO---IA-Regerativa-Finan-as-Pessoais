🚀 Desafio: Curadoria de Conhecimento e Aprendizagem Ativa com NotebookLM
Este projeto foi desenvolvido como parte de um desafio da DIO (Digital Innovation One), focado em utilizar o NotebookLM (IA do Google) como uma ferramenta de apoio à aprendizagem ativa e organização de informações complexas.

🎯 Objetivo do Projeto
O foco foi criar um Caderno Temático de Educação Financeira, focado em organização familiar e reserva de emergência. O objetivo principal foi transformar fontes técnicas e densas (como manuais do Banco Central e CVM) em um guia de estudo acessível, utilizando técnicas de Prompt Engineering para extrair o máximo de valor da IA.

📚 Fontes Utilizadas
Para este caderno, foram selecionadas fontes oficiais e abertas:

Banco Central do Brasil: Caderno de Gestão de Finanças Pessoais.

CVM (Comissão de Valores Mobiliários): Guia de Planejamento Financeiro.

ENEF (Estratégia Nacional de Educação Financeira): Orientações sobre Reserva de Emergência.

🛠️ Funcionalidades e Metodologia
Curadoria de Fontes: Seleção e upload de documentos PDF/Texto para alimentar a base de conhecimento da IA.

Prompt Engineering: Teste de diferentes abordagens (Prompts Diretos, Persona e Estruturados) para validar a precisão das respostas e referências.

Aprendizagem Ativa: Uso de perguntas estratégicas para gerar insights práticos sobre economia doméstica.

📦 Resultados Obtidos (O Miniguia)
O projeto resultou em um repositório de conhecimento contendo:

Resumo Estruturado: Uma visão geral sobre como organizar o orçamento familiar do zero.

Glossário Técnico: Termos essenciais como SELIC, Juros Compostos, Liquidez e Gastos Fixos/Variáveis.

Biblioteca de Prompts Reutilizáveis: Comandos prontos para futuras consultas ou revisões de conteúdo financeiro.

Dicas para o preenchimento na DIO:
Título do Projeto: Aprendizagem Ativa com NotebookLM: Guia de Organização Financeira Familiar

Tecnologias Utilizadas: NotebookLM, Google Gemini, Generative AI, Prompt Engineering, Curadoria de Conteúdo.

Principais Aprendizados: Destaque que a IA não apenas resume, mas ajuda a "conectar os pontos" entre diferentes fontes, agilizando o processo de estudo e aplicação prática.
